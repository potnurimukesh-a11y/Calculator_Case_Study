Calculator Program using Tkinter

📌 Overview

This project is a GUI-based calculator application developed using Python’s Tkinter framework. It is part of a case study based on concepts learned during classroom sessions.

The application enables users to perform basic arithmetic operations through an interactive graphical interface.

---

🎯 Objectives

- To design a graphical user interface using Tkinter
- To implement arithmetic operations in Python
- To handle user input using event-driven programming
- To integrate logic with GUI components

---

📂 Project File

- "Calculator_Program_using_tkinter.py" → Main Python file containing the calculator implementation

---

⚙️ Features

- Simple and user-friendly interface
- Supports:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Button-based input system
- Display screen for results
- Clear/reset functionality

---

🛠️ Technologies Used

- Python
- Tkinter (Standard GUI library in Python)

---

🧠 Concepts Applied

- Event handling
- GUI widgets (Buttons, Entry)
- Layout management
- Function binding

---

▶️ How to Run

1. Install Python (if not already installed)
2. Run the program:

python Calculator_Program_using_tkinter.py

---

📷 Output

The program opens a calculator window where users can perform basic calculations using on-screen buttons.

---

📚 Conclusion

This project demonstrates how Python can be used to build interactive desktop applications using Tkinter. It highlights the combination of user interface design and functional logic.
