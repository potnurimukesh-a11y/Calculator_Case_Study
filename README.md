Calculator Case Study (Tkinter)

📌 Overview

This project is a calculator application developed using Python’s Tkinter framework. It is part of a case study based on concepts learned during classroom sessions.

The application provides a simple graphical interface for performing basic arithmetic operations.

---

🎯 Objectives

- To understand GUI development using Tkinter
- To implement basic arithmetic operations
- To handle user input through an interactive interface
- To apply event-driven programming concepts

---

⚙️ Features

- User-friendly graphical interface
- Supports basic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Button-based input system
- Display screen for input and results
- Clear/reset functionality

---

🛠️ Technologies Used

- Python
- Tkinter (GUI Framework)

---

🧠 Concepts Applied

- Event-driven programming
- GUI design using widgets (Buttons, Entry, Labels)
- Layout management
- Input handling and validation

---

▶️ How to Run

1. Make sure Python is installed on your system
2. Run the Python file:

python calculator.py

---

📷 Output

The application opens a calculator window where users can perform arithmetic operations using buttons.

---

📚 Conclusion

This project demonstrates how Python can be used to build interactive GUI applications. It highlights the integration of logic and interface design using Tkinter.
